# CYBERBEANS 苹果风格网站项目大纲

## 项目概述
重新设计CYBERBEANS独立站，采用苹果官网的极简主义设计风格，提供更专业、更优雅的用户体验。

## 网站结构

### 1. 首页 (index.html)
**目标**: 品牌印象建立，产品亮点展示，用户引导

**苹果风格设计元素**:
- **导航栏**: 苹果风格半透明导航栏，悬停下拉菜单
- **英雄区域**: 大幅产品展示图，简洁的品牌标语
- **产品展示**: 苹果风格的产品网格布局
- **特色功能**: 突出显示核心产品特点
- **品牌故事**: 简洁的品牌价值主张

**内容结构**:
- **导航栏**: Logo + 车型菜单 + 其他菜单
- **英雄区域**: 主产品图 + 品牌标语 + CTA按钮
- **产品分类**: 按车型分类的产品展示
- **特色功能**: 创新技术、精工品质、用户至上
- **最新产品**: 苹果风格的产品轮播

### 2. 产品页面 (products.html)
**目标**: 完整产品目录展示，智能筛选，购买转化

**苹果风格设计元素**:
- **导航栏**: 与首页一致的导航体验
- **产品网格**: 苹果商店风格的产品展示
- **筛选器**: 侧边栏筛选，实时更新
- **产品卡片**: 苹果风格的产品信息展示

**内容结构**:
- **页面标题**: 简洁的页面描述
- **筛选侧边栏**: 车型、类别、价格筛选
- **产品网格**: 响应式产品卡片布局
- **产品对比**: 苹果风格的产品对比工具
- **分页导航**: 优雅的分页设计

### 3. 关于我们页面 (about.html)
**目标**: 品牌故事讲述，建立信任，差异化定位

**苹果风格设计元素**:
- **简洁布局**: 苹果风格的页面布局
- **时间线**: 品牌发展历程的时间线展示
- **团队介绍**: 专业的团队展示
- **价值主张**: 清晰的品牌价值展示

**内容结构**:
- **品牌故事**: CYBERBEANS创立背景
- **核心价值**: 创新、品质、用户至上
- **发展历程**: 时间线展示品牌成长
- **团队介绍**: 核心团队成员
- **认证资质**: 专业认证展示

### 4. 支持页面 (support.html)
**目标**: 客户服务支持，多渠道联系，售后保障

**苹果风格设计元素**:
- **联系方式**: 多种联系方式展示
- **FAQ**: 苹果风格的常见问题解答
- **服务流程**: 清晰的服务流程图
- **在线支持**: 实时客服聊天功能

**内容结构**:
- **联系信息**: 电话、邮箱、在线客服
- **FAQ**: 常见问题解答
- **服务流程**: 专业的服务流程
- **在线支持**: 实时聊天工具
- **公司信息**: 地址、工作时间等

## 技术实现

### 前端技术栈
- **HTML5**: 语义化标记，SEO友好
- **CSS3**: Flexbox和Grid布局，响应式设计
- **JavaScript**: 原生JS实现交互，无框架依赖
- **动画库**: Anime.js实现流畅动画

### 核心功能
1. **苹果风格导航栏**: 悬停下拉菜单
2. **产品筛选系统**: 多维度筛选
3. **产品对比工具**: 详细对比功能
4. **购物车系统**: 完整的购物流程
5. **响应式设计**: 适配所有设备

### 视觉效果库
- **Anime.js**: 流畅的动画效果
- **Splide.js**: 苹果风格的轮播组件
- **ECharts.js**: 数据可视化
- **PIXI.js**: 高性能视觉特效

### 色彩系统
- **主背景**: 纯白 (#FFFFFF)
- **文字主色**: 深空灰 (#1D1D1F)
- **链接颜色**: 蓝色 (#0066CC)

- **主标题**: SF Pro Display / Noto Sans SC Bold
- **正文**: Noto Sans SC Regular
- **装饰**: ZCOOL KuaiLe (品牌名称)

### 布局规范
- **容器宽度**: 最大1200px
- **网格系统**: 12列网格
- **间距**: 8px基础网格
- **圆角**: 12px、18px统一圆角
